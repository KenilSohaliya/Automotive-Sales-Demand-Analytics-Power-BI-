# Automotive Sales & Demand Analytics (Power BI)

End-to-end analytics project using **Python + Power BI** to analyze automotive demand trends in the German market.  
Focus areas include **EV adoption**, **pricing trends**, **brand performance**, and **CO₂ emissions** to support data-driven planning and sustainability insights.

## Project Highlights
- Built a complete analytics workflow: data prep (Python) → modeling → dashboard (Power BI)
- Tracked key market signals: EV adoption, average prices, brand comparisons, emissions trends
- Combined market data with emissions data to add sustainability context
- Designed interactive visuals for fast exploration and decision support

## Tools & Skills
- **Power BI** (DAX, Power Query, data modeling)
- **Python** (Pandas, NumPy, data cleaning, EDA)
- Visualization and KPI storytelling
